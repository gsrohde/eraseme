Creates a dynamical_system object from the differential quantities, parameters, drivers, and modules, and then uses the system object to determine the derivatives of the differential quantities at the specified time.

Parameters
[in]differential_quantitiesAn R list of named elements representing the current values of the differential quantities
[in]timeAn R numeric vector with one element specifying the time index
[in]parametersAn R list of named elements representing the parameters
[in]driversAn R data frame representing the time series of the drivers
[in]direct_module_namesAn R string vector of the names of the direct modules
[in]differential_module_namesAn R string vector of the names of the differential modules
Returns
An R list of named elements representing the derivatives of the differential quantities
