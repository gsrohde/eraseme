Detailed Description

Various physical constants are collected here.

These physical constants may be subdivided according to how "constant" they are:

  • stefan_boltzmann and ideal_gas_constant: These are truly constant, and as of the 2019 redefinition of SI base units, they have exact numerical values when expressed in SI units.
  • molar_mass_of_water: This depends on the relative frequency of the water's isotopologues, but for practical purposes, and because this composition doesn't usually vary greatly, we treat it as constant with the approximate value given.
  • atmospheric_pressure_at_sea_level: More precisely, this is standard sea-level pressure, which has the value given here by definition and represents the near-average atmospheric pressure at sea level on the Earth. The actual atmospheric pressure at any sea-level location changes continuously over time and may vary considerably from this.
