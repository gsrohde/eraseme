SEXP R_evaluate_module ( SEXP  mw_ptr_vec,
SEXP  input_quantities 
)

Determines the values of a module's output quantities from the supplied values of its input quantities.

Parameters
[in]mw_ptr_vecA single-element vector containing one R external pointer pointing to a module_wrapper_base object, typically produced by the R_module_wrapper_pointer() function. If the vector has more than one element, only the first will be used.
[in]input_quantitiesA list of named numeric elements where the name of each element corresponds to one of the module's input quantities. Any element whose name does not correspond to one of the module's input quantities will be ignored by the module.
Returns
A list of named numeric elements where the name of each element corresponds to the one of the module's output quantities
